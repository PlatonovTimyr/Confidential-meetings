# 🔒 Confidential Meetings

**Максимально защищённые P2P видеовстречи без серверов**

[![Лицензия](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![GitHub Pages](https://img.shields.io/badge/demo-GitHub%20Pages-brightgreen)](https://timyr.github.io/confidential-meetings/)

## 🛡️ Возможности

- **End-to-End шифрование** — AES-256-GCM + ECDH P-521
- **P2P соединение** — видео и аудио напрямую между участниками
- **Без серверов** — никакие данные не проходят через третьи стороны
- **Детектор вторжений** — обнаружение MITM, replay-атак, аномалий
- **Цифровые подписи** — HMAC-SHA512 для каждого кадра
- **Защищённый чат** — все сообщения шифруются
- **Демонстрация экрана** — с шифрованием
- **QR-коды** — для быстрого подключения
- **Система тревог** — красный экран при обнаружении угрозы
- **Адаптивный дизайн** — работает на всех устройствах

## 🔐 Шифрование

### Используемые алгоритмы:
- **AES-256-GCM** — шифрование видео и аудио потоков
- **ECDH P-521** — обмен ключами
- **HMAC-SHA512** — цифровые подписи каждого кадра
- **SHA-256** — проверка целостности данных

### Защита от атак:
- ✅ Replay-атаки
- ✅ MITM (Man-in-the-Middle)
- ✅ Подмена данных
- ✅ Временные аномалии
- ✅ Нарушение последовательности кадров

## 🚀 Быстрый старт

### Создание встречи:
1. Откройте [приложение](https://timyr.github.io/confidential-meetings/)
2. Нажмите "Создать встречу"
3. Введите имя и настройте камеру/микрофон
4. Поделитесь ссылкой или QR-кодом

### Подключение к встрече:
1. Перейдите по ссылке или отсканируйте QR-код
2. Введите имя
3. Нажмите "Подключиться"

## 💻 Локальный запуск

```bash
# Клонируйте репозиторий
git clone https://github.com/Timyr/confidential-meetings.git

# Перейдите в папку
cd confidential-meetings

# Запустите локальный сервер
python -m http.server 8000

# Откройте в браузере
http://localhost:8000
```

## 🔒 Приватность

- **Нет серверов** — только статический хостинг
- **Нет базы данных** — никакие данные не хранятся
- **Нет регистрации** — не требуется создавать аккаунт
- **Нет логов** — встречи полностью анонимны
- **Ключи в URL** — передаются только участникам

## ⚠️ Система тревог

При обнаружении аномалии:
1. Экран становится красным
2. Отображается предупреждение с деталями
3. Автоматическое отключение через 15 секунд
4. Возможность принять или завершить встречу
